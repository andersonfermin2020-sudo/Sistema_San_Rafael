# Sistema de Gestión Hospitalaria "San Rafael"

Sistema en consola para la gestión de personal, pacientes, consultas, inventario y facturación del Hospital General "San Rafael".

## Características

- Gestión de personal médico y administrativo
- Gestión de pacientes y consultas
- Inventario de medicamentos con alertas
- Facturación automática
- Reportes por rol

## Requisitos

- Python 3.8 o superior
- Sistema operativo: Windows, Linux o macOS

## Instalación

1. Clonar el repositorio
2. No requiere instalación de dependencias externas (solo librerías estándar de Python)

## Ejecución
```bash
python main.py
```

## Estructura del Proyecto

- `src/models/` - Clases del modelo (POO)
- `src/controllers/` - Lógica de negocio
- `src/views/` - Interfaz de usuario (menús)
- `src/utils/` - Utilidades y validaciones
- `data/` - Almacenamiento en JSON
